HTML Basics

These flashcards drill you on HTML basics with code snippet examples.


HTML
Hypertext Markup Language is a markup language rather than a programming language, which means it defines how a page will be presented rather than how elements behave. It holds all the instructions for how the web page is displayed by ordering elements that enclose the content.
CSS
Cascading Style Sheets is a language that controls the styling of the page and how it appears. It can be added or linked in the HTML style tag, and its benefit is its flexibility. CSS isn't necessary to build a webpage, but utilizing it allows design elements to be clumped and repeated throughout the layout with no inline, repetitive style attributes.
element
Elements are the individual building blocks of HTML. Most have a beginning tag (<p>) and an ending tag(</p>). A few tags like <img> and <br> are called empty tags and only have a starting tag.
Elements of a Basic HTML doc
<!DOCTYPE html> - This says this document is using HTML5
<html lang="en"> - The root element. All page elements are nestled in it
<head> - holds all the (meta) information about the page. Not visible to viewer
<meta charset="UTF-8"> - Allows for easy storage of ascii (English and number) characters, but will also support characters for other languages
<meta name="viewport" content="width=device-width, initial-scale=1.0"> - Sets the browser width (probably) to screen width. Introduced with Safari to help with mobile viewing More viewport tag info
<title>Document</title> - Subject of the document contents that also functions as the suggested bookmark title
</head>
<body> - Contains all elements that are visible
</body>
</html>

tags
One half of the element. An opening tag has a beginning angle bracket(<), the content (i.e. p or div), and an ending angle bracket with a forward slash(/>).
    Type Examples:
  • <p> - paragraph
  • <div> - general divider/grouping of elements. Most widely used type of element
  • <h1> - heading tags make larger text by default than div or p. There are h1 through h6 with h1 being the largest
  • <a> - anchor tags hold links
  • <img> - image tags hold images
attributes
Give extra information about the element they live in. They always appear in the starting tag of the element they belong to. In <a href="google.com"<Google link</a>, <a> is the anchor element to hold a link, href="" is the attribute that holds the destination URL, and Google link is the inner HTML that will be the hyperlink.
